    StopMode

    StopMode indicates whether to simulate orderly (administrative) or chaotic (power off) shutdown

    Name Number Description
    ORDERLY_STOP 0
    CHAOTIC_STOP 1

    DeviceService

    DeviceService provides means to control inventory of simulated devices (switches and IPUs) and their ports

    Method Name Request Type Response Type Description
    GetDevices GetDevicesRequest GetDevicesResponse GetDevices gets a list of all simulated devices (switches and/or IPUs)
    GetDevice GetDeviceRequest GetDeviceResponse GetDevice gets a specific device entry
    AddDevice AddDeviceRequest AddDeviceResponse AddDevice creates a new simulated deviceand start its P4Runtime and gNMI services
    RemoveDevice RemoveDeviceRequest RemoveDeviceResponse RemoveDevice removes a simulated device
    StopDevice StopDeviceRequest StopDeviceResponse StopDevice stops the simulated deviceP4Runtime and gNMI services
    StartDevice StartDeviceRequest StartDeviceResponse StartDevice starts the simulated deviceP4Runtime and gNMI services
    DisablePort DisablePortRequest DisablePortResponse DisablePort disables the specified port
    EnablePort EnablePortRequest EnablePortResponse EnablePort enables the specified port
    EmitLLDPPacket EmitLLDPPacketRequest EmitLLDPPacketResponse EmitLLDPPacket emits the specified LLDP packet on a given device port.
