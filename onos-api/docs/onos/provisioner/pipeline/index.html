    ProvisionerService

    ProvisionerService allows managing inventory of various device configuratoins, e.g., pipeline, chassis.

    Method Name Request Type Response Type Description
    Add AddConfigRequest AddConfigResponse Add registers new pipeline configuration
    Delete DeleteConfigRequest DeleteConfigResponse Delete removes a pipeline configuration
    Get GetConfigRequest GetConfigResponse Get returns pipeline configuration based on a given ID
    List ListConfigsRequest ListConfigsResponse stream List returns all registered pipelines
