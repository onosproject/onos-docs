    UEService

    UEService provides an API for managing UEs and various aspects of information associated with UEs.

    Method Name Request Type Response Type Description
    CreateUE CreateUERequest CreateUEResponse Create a new UE entity and its initial set of aspects.
    GetUE GetUERequest GetUEResponse Get a UE entity populated with the requested aspects.
    UpdateUE UpdateUERequest UpdateUEResponse Update an existing UE entity populated with the requested aspects. Only the aspects present in the UE entity will be updated; others will be left unmodified. New aspects can be added via update.
    DeleteUE DeleteUERequest DeleteUEResponse Delete the specified aspects of a UE entity.
    ListUEs ListUERequest ListUEResponse stream ListUEs returns a stream of UE entities populated the requested aspects.
    WatchUEs WatchUERequest WatchUEResponse stream WatchUEs returns a stream of UE change notifications, with each UE populated with only the requested aspects.
