    ChannelMeta

    ChannelMeta contains metadata relating to a subscription channel

    Field Type Label Description
    app_id string app_id is an identifier for the application that created the channel shared across all instances of the application
    app_instance_id string app_instance_id is an identifier unique to a single node within the application that created the channel
    e2_node_id string e2_node_id is the identifier for the E2 node to which the subscription is targeted
    transaction_id string transaction_id is a persistent identifier for the transaction
    subscription_id string subscription_id is the identifier for the underlying subscription
    service_model ServiceModel service_model is the service model specification
    encoding Encoding encoding indicates the type of encoding used within the parent message
    revision uint64
    finalizers string repeated
